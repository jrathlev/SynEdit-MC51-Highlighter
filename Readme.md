### SynHighlighterMC51xx

This unit provides a **[SynEdit](https://github.com/SynEdit/SynEdit)
-Highlighter** for *51-family microcontroller* assembler.
To use it for a specific member of this family the appropriate special
function registers (SFR) definitions must be loaded 
using the Highlighter function **InitHighlighter(McuName)**.
McuName (*filename.mcu*) is a text file also used as include file in 
assembly language.

A collection of 237 mcu files for the most common µC types is included.

last updated: September 2019
